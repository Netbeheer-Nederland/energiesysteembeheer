# Definities

*Deze pagina is een placeholder.*

We nemen een begrip alleen op als deze binnen de context van de netbeheerder een unieke betekenis heeft, zoals `aansluiting`. Begrippen zoals `postcode` nemen we niet op.

Elk begrip moet uniek zijn binnen de context van de netbeheerder, wat betekent dat er per (voorkeurs)term maar één definitie bestaat. Worden in het dagelijkse spraakgebruik meerdere definities gehanteerd, dan geldt de volgende volgorde van gewicht:

1. definities uit regelgeving krijgen voorrang
2. daarna prioriteren wij de buitenstaander boven het interne taalgebruik
3. intern krijgen de definities die verband houden met onze kernactiviteiten voorrang
4. als laatste op deze ladder staat de generieke binnenwereld van de netbeheerder (IT, HR, Finance)

De naam van een begrip volgt de Nederlandse spellingsregels en begint niet met een hoofdletter, tenzij dat de juiste spelling van het woord is.

Verder geldt:

- Schrijf de definitie zo dat het begrip in een zin kan worden vervangen door de definitie
- Gebruik genus-differentia: benoem het type en het onderscheid.
- Vermijd circulaire verwijzingen.
- Vermeld bij voorkeur een bron.
- Geef context alleen als die essentieel is voor betekenis.
